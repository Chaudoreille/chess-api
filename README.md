# **chess-api**

An API handling chess moves.

## Install

-   install npm

    ```shell
    $ sudo apt install npm
    ```

-   install project

    ```shell
    $ npm install
    ```

-   create a .env file
    ```shell
    $ cp .env.example .env
    ```
    setup environmental variables in .env

## Run server

    ```shell
    $ npm start
    ```
